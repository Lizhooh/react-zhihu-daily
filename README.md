## Use

``` bash
# install dependencies
npm install

# before, run server
cd ./server
npm install
npm start

# after
cd ../

# serve with hot reload at localhost:3000
npm start

# build for production with minification
npm run build
```


### Add sass/less

``` bash
cd node_modules/react-scripts
npm install --save sass-loader
npm install --save less-loader
```
