## react-zhihu-daily
基于 React.js 实现的 Webapp 知乎日报 (非官方)

![](/demo/demo.gif)

### install

``` bash
# install dependencies
npm install

# before, install nodejs server
cd ./server
npm install
```

### Use
在 Chrome 上使用手机预览模式（F12 => ctrl + shift + m）

```bash
# 先启动 nodejs 服务
npm run server

# 再启动 Webapp，在 localhost:3000
npm start

# 编译整个应用
npm run build
```
